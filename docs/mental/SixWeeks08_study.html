# 実践手法

全て有料枠でしたが，以下のことを実践します．

  • 現在と異なる人間関係の構築
  • 英語の勉強(英語論文を読む)
  • 統計・機械学習の数学の勉強
  • 以下の能力を伸ばす
    • コードの直感・解読・改造
    • パロディ
    • 興味のある分野での単純作業(動画編集等)
  • 一旦，無茶な成果を考えてそれまで熱中する
  • 自分の肩書き・役割を考えない
