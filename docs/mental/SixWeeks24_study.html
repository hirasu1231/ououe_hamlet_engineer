# 最高の性格

  • 人は自分の性格についてわかっていない
  • 自分の性格を完全に変えるのはムリ
  • 最高の性格=悩みが無く幸せな性格
  • 最高の性格に近づける

# 7つの神性格

  1. 誠実性
  • 誠実性は高ければ高い方が良い
  • 強迫性障害に気を付ける
  1. 神経症的傾向
  • 神経症的傾向は少ない方が良い
  • 出来るだけ改善しておく
  • 瞑想（MBSR）　運動　If Thenルールを使う
  1. 開放性
  • 開放性は感情の受容の高さが大事
  • 新しい事への恐怖を受け入れる強さ
  1. 外向性
  • 外向性はポジティブ感情を高める
  • みんなと仲良くする必要はない
  • 他人と関わる時にポジティブな感情を高める
  • ポジティブ感情を高めるトレーニングをする
  • お勧め本：脳科学は人格を変えれるか
  1. 協調性
  • 率直さが大事
  • 自分にも他人にも嘘をつかない事
  • Noと言えるようになる
  • アサーションが大事 ※アサーション:自分の意見を相手の立場を尊重しながらもしっかり伝えるコミュニケーションスキル

# 神性格を持つ人とは

「自尊心が高く自分をポジティブに見る事ができ，強いアイデンティティを持ち楽観性があり，自信を持って生きる事が出来る」

# 幸福度をブーストさせる2つの性格

  1. 熱意(外向性のサブカテゴリー)
  2. 自閉性の改善(神経症的傾向のサブカテゴリー)

# ビッグファイブのサブカテゴリー分け

  1. 外向性
  • 熱意：達成に仲間と喜びを分かち合える(友好的で感情表現豊か)
  • 自己主張：支配的で報酬にモティベーションを感じるか(自分を発信)
  1. 神経症的傾向
  • 自閉性：不安・抑うつ
  • 変動性：怒りやいらいら，気分のアップダウン
  1. 誠実性
  • 勤勉さ：目標に向かってコツコツ進む
  • コントロール：決まり事を守る，社会的な真面目さ
  1. 協調性
  • 慈悲：他人の感情的に共感できるか
  • 礼儀：他人に礼節を持って接する事，周りに合わせる礼儀
  1. 開放性
  • 美的好奇心：アーティスト
  • 知的好奇心：DaiGo・研究者

# 幸福度を決めるサブカテゴリー

1位　熱意（⊂外向性）
2位　自閉性が低い(⊂神経症的傾向)
3位　勤勉さ(⊂誠実性）
4位　慈悲(⊂協調性)
5位　知的好奇心(⊂開放性)

# 幸福度と全く関係の無い5つの性格

1位　礼儀(⊂協調性)
2位　統制(⊂誠実性)
3位　変動性(⊂神経症的傾向)
4位　美的好奇心(⊂開放性)
5位　自己主張(⊂外向性)

# 性格を変える方法

  • If Thenプランニング（ルール）に落とし込む
  • 行動を変えると性格がある程度変わる

# 成功する性格はユーモア

  • 成功には知性より性格
  • 誠実性と開放性が大事
  • 誠実性は瞑想とIf Thenルール

# ユーモアスタイルを変える（外向性と開放性）

笑いを取る時の4つのユーモア

  1. 親和的ユーモア
  • 誰もが知っているユーモア
  • あるあるネタ・変顔
  1. 自己高揚的ユーモア
  • ストレスフルや何気ない日常の中を楽しくする
  • 面白くない物事を遊び心を使って変換する力
  • 自分の失敗・黒歴史を前向きに明るく話す
  1. 攻撃的ユーモア
  • 他人をディスって笑いを取る
  1. 自己卑下的ユーモア
  • 自虐ネタ
  • 駄目な自分を見せて受け入れて貰う
  • 前向きにさせる自己高揚的ユーモアとは違う

# 4つのユーモアスタイルの特徴

  1. 親和的ユーモア
  • 友達が多く感情が安定している
  • 外向性・開放性が高い
  1. 自己高揚的ユーモア
  • うつや不安傾向が少ない
  • 自信があり楽観性が高い
  • 幸福度と相関が高い
  • 異性からモテやすくなる
  1. 攻撃的ユーモア
  • ブラックジョークが多い人はメンタルが弱い
  • 協調性と誠実性は低い
  1. 自己卑下的ユーモア
  • メンタルが弱く協調性が低く誠実性が低い

# ユーモアスタイルを変えるには

  • 自分のユーモアスタイルを知る
  • 自己高揚的なユーモアに切り替える
  • しんどい時にもポジティブを見出す
  • しんどい現実を受け入れて面白い所をみつける ※If Thenルールで改善する

# フロー状態（没頭状態）に入る性格

  • 人間の脳はフローに入りにくく作らている
  • 没頭すると周りが見えなくなって危険であった
  • フロー状態は疲れやすいという理由から

# フローに入りやすい性格

  1. 誠実性が高い人
  2. 神経症的傾向が低い
  3. 他の性格は関係無い ※誠実性と神経症的傾向は改善し易い

# フローに入りやすい行動

  • 今の自分には少しチャレンジになる事
  • 出来るかな？という不安感が必要
  • メンタルが弱いと気になってフローが途切れやすい
  • 知性とフロー状態には相関が無い

# 実践手法

以下の手法を実践します．

  • 「If Thenルール (opens new window)」を習慣づける
  • if-thenプランニング徹底解説〜効果量0.99の単独で人生変わる最強心理テク (opens new window)を確認
  • 開放性(知的好奇心)を高める
    • 若くても成功するための【好奇心の鍛え方 (opens new window)
  • 自己卑下的ユーモア -> 自己高揚的ユーモア
  • フロー状態（没頭状態）に入る練習をする
    • 今の自分*1.1のことにチャレンジする
