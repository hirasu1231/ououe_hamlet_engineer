# インストール

tsutomu7/nim (opens new window)のイメージをdocker-composeでPullします．
僕はdocker-compose.ymlを以下のように記述しました．

version: "3"
services:
  jupyter:
    image: tsutomu7/nim:latest
    # user: root
    volumes:
      - ./:/root/work
    ports:
      - "88:8888" # for Jupyterlab

コンテナが立ち上がったら http://localhost:88/ にアクセスすると Jupyter Lab が開きます．

$ mkdir jupyter-nim
$ cd jupyter-nim
$ touch docker-compose.yml
# 中身記述後
$ docker-compose up
