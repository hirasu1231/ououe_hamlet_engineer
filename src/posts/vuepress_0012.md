## ファイル構成
```
.vuepress
└── public
    └── /audio/audio_example/
        └── drums.wav
```

## 動画の貼り付け
以下のコードを動画を貼りたい場所に記述します．
```md
<audio src="/audio/audio_example/drums.wav" controls></audio>
```

<audio src="/audio/audio_example/drums.wav" controls></audio>
